Conectar ao Google Sheets

Este dashboard puxa os dados de compra de gado diretamente de uma planilha no Google Drive. Configure uma vez e ele atualiza automaticamente.

  1. Abra a planilha FRIOESTE_COMPRA_GADO no Google Sheets
  2. Vá em Arquivo → Compartilhar → Publicar na web
  3. Selecione a aba "BASE" e formato "Valores separados por vírgulas (.csv)"
  4. Clique em Publicar e copie o link gerado
  5. Cole o link abaixo:

O link deve ser algo como: https://docs.google.com/spreadsheets/d/e/2PACX.../pub?output=csv

Alternativa: Se preferir, carregue o arquivo Excel direto do computador:
